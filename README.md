# Port Scanner

A simple Python port scanner built using the socket module.

## Features

* Scan TCP ports
* Detect open ports
* Uses Python socket programming
* Beginner-friendly implementation

## Technologies Used

* Python
* Socket Programming

## How to Run

```bash
python scanner.py
```

## Example Output

```text
Scanning target: 127.0.0.1

Port 80 is OPEN

Scan completed.
```

This project is for educational purposes only.

Only scan systems that you own or have permission to test.
